How to get Client ID + Token →
  1. Register an app at dev.twitch.tv/console. Redirect URL = http://localhost. Type = Public. Copy the Client ID.
  2. Paste this in your browser (replace CLIENT_ID):
    https://id.twitch.tv/oauth2/authorize?client_id=CLIENT_ID&redirect_uri=http://localhost&response_type=token&scope=chat:read
  3. Twitch redirects to http://localhost/#access_token=XXXX. Page fails to load (fine) — copy the token from the URL bar.

OBS Browser Source — custom styling →
OBS lets you add CSS in the Browser Source dialog (the "Custom CSS" field). Targets:
  • #info — the overlay container
  • .ch — channel-name line · .ti — title line · .me — meta line
  • #bar — purple progress bar
  • video — the player itself
Example — bigger title, gold bar, hide everything but title:
.ch,.me{display:none} .ti{font-size:32px;color:#ffd700} #bar{background:#ffd700;height:6px}
